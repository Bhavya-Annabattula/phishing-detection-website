About the Project

AI-Powered Phishing Detection System is a cybersecurity solution that uses Machine Learning and Natural Language Processing (NLP) techniques to identify phishing websites. The model is trained on over 109,000 URLs and analyzes URL patterns to determine whether a website is legitimate or malicious.

This project helps users detect online threats in real time, reducing the risk of credential theft, financial fraud, and malicious website attacks. Built with Python, Flask, Scikit-learn, and Tailwind CSS, it demonstrates the practical application of Artificial Intelligence in cybersecurity.
